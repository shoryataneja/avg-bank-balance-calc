# Frontend — Average Bank Balance Calculator

This is the React frontend for the Average Bank Balance Calculator.

**For full setup and run instructions, see the [root README](../README.md).**

## Quick Start

```bash
npm install
npm run dev
```

Runs on **http://localhost:5173** by default.  
Requires the backend to be running on **http://localhost:8000**.

## Tech

- React 19 + Vite 8
- Tailwind CSS 3
- Recharts 3
